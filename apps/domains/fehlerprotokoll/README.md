# Fehlerprotokoll Domain Pack

This domain pack contains example rules and placeholders for generating sample
Fehlerprotokoll documents. It is separate from core OCR/LayoutLM.

- `rules.yaml`: regex rules
- `plugins.py`: domain-specific plugins/validators
- `generate.py`: dummy generator (text placeholder)
- `sample_docs/`: place sample PDFs/images
- `expected/`: expected JSON/CSV outputs for regression tests
